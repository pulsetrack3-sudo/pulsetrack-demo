# PulseTrack (Static Site)

This is a simple, single-page website for **PulseTrack — Junk Removal Leads + Tracking**.

## What it does
- Landing page + pricing
- "Request a Demo" modal
- Netlify Forms integration (captures demo requests)

## How to use (GitHub)
1. Create a new GitHub repository named `pulsetrack`
2. Upload these files:
   - `index.html`
   - `netlify.toml`
   - `README.md`

## Deploy (Netlify)
1. Log into Netlify
2. Add new site → Import from Git → choose your GitHub repo
3. Deploy

## Netlify Forms
This repo uses Netlify Forms.
After your site is deployed:
- Netlify dashboard → Site → Forms
- You will see demo submissions there.
